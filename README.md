## Как запустить проект

1. Скачай и установи [Ren'Py SDK](https://www.renpy.org/latest.html) (рекомендуется последняя стабильная версия).
2. Запусти Ren'Py Launcher.
3. Выбери **"Choose Project / Add Project"** и укажи папку с этим репозиторием.
4. Выбери проект в списке и нажми **"Launch Project"**.

Проект готов к запуску без дополнительных зависимостей (чистый Ren'Py).
